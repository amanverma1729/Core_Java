# ☕ Core Java Mastery

A structured and comprehensive repository documenting my journey of mastering **Core Java** from beginner to advanced concepts.

This repository contains detailed notes, coding exercises, hands-on projects, interview preparation, and practical implementations of Java concepts. It is designed as a complete learning resource for anyone who wants to build a strong foundation in Java programming and Object-Oriented Programming (OOP).
